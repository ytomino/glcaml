Module Win

module Win: sig .. end
Provides a few basic functions to use OpenGL from Ocaml Graphics module
val init_opengl : unit -> unit
Set up Ocaml graphics window for OpenGL input. A window has to be created before this function may be called. If the window had a title it will have to be set again
val swap_buffers : unit -> unit
Swap the back buffer to the screen
val usleep : int -> unit
Sleep for n microseconds
